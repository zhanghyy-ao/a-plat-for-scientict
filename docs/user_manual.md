# 计算机实验室管理系统 - 用户手册

## 1. 系统简介

计算机实验室管理系统是一个综合性平台，用于管理实验室的学生、导师、项目、新闻和成果等信息。系统支持管理员对页面内容的修改，导师对学生的管理，以及学生对个人信息的查看和更新。

### 1.1 系统功能

- **用户认证**：登录、登出功能
- **学生管理**：添加、编辑、删除学生，分配导师
- **导师管理**：添加、编辑、删除导师
- **课题进度**：学生提交进度，导师审阅和反馈
- **内容管理**：管理新闻、成果、关于我们和联系方式

### 1.2 用户角色

- **管理员**：拥有系统的所有权限，可以管理所有内容
- **导师**：可以查看和管理自己的学生，审阅学生进度
- **学生**：可以查看个人信息，提交课题进度

## 2. 系统登录

### 2.1 登录页面

1. 打开浏览器，访问系统 URL
2. 进入登录页面，输入用户名和密码
3. 点击"登录"按钮

### 2.2 默认账号

- **管理员**：用户名 `admin`，密码 `admin123`
- **导师**：用户名 `mentor1`，密码 `mentor123`
- **学生**：用户名 `student1`，密码 `student123`

## 3. 管理员操作指南

### 3.1 学生管理

1. 登录系统后，点击左侧导航栏中的"学生管理"
2. 在学生管理页面，可以查看所有学生的列表
3. 点击"添加学生"按钮，填写学生信息并提交
4. 点击学生列表中的"编辑"按钮，修改学生信息
5. 点击学生列表中的"删除"按钮，删除学生
6. 点击学生列表中的"分配导师"按钮，为学生分配导师

### 3.2 导师管理

1. 登录系统后，点击左侧导航栏中的"导师管理"
2. 在导师管理页面，可以查看所有导师的列表
3. 点击"添加导师"按钮，填写导师信息并提交
4. 点击导师列表中的"编辑"按钮，修改导师信息
5. 点击导师列表中的"删除"按钮，删除导师

### 3.3 内容管理

1. 登录系统后，点击左侧导航栏中的"内容管理"
2. 在内容管理页面，可以选择要管理的内容类型：
   - 新闻管理
   - 成果管理
   - 关于我们管理
   - 联系方式管理
3. 对于每种内容类型，可以添加、编辑、删除相应的内容

## 4. 导师操作指南

### 4.1 我的学生

1. 登录系统后，点击左侧导航栏中的"我的学生"
2. 在我的学生页面，可以查看自己指导的学生列表
3. 点击学生列表中的学生，查看学生的详细信息和进度历史

### 4.2 待审进度

1. 登录系统后，点击左侧导航栏中的"待审进度"
2. 在待审进度页面，可以查看学生提交的待审阅进度
3. 点击进度列表中的进度，查看进度详情
4. 填写反馈意见，选择评分和是否通过，点击"提交反馈"按钮

### 4.3 个人信息

1. 登录系统后，点击右上角的用户头像，选择"个人信息"
2. 在个人信息页面，可以查看和修改个人信息

## 5. 学生操作指南

### 5.1 课题进度

1. 登录系统后，点击左侧导航栏中的"课题进度"
2. 在课题进度页面，可以查看自己的进度历史
3. 点击"提交进度"按钮，填写进度信息并提交
4. 点击进度列表中的进度，查看进度详情和导师反馈

### 5.2 个人信息

1. 登录系统后，点击右上角的用户头像，选择"个人信息"
2. 在个人信息页面，可以查看和修改个人信息

## 6. 系统功能详细说明

### 6.1 学生管理

#### 6.1.1 添加学生

- **操作步骤**：
  1. 进入学生管理页面
  2. 点击"添加学生"按钮
  3. 填写学生信息：用户名、密码、姓名、学号、性别、年级、学生类型、专业、研究课题、入学日期
  4. 点击"提交"按钮

- **注意事项**：
  - 用户名和学号必须唯一
  - 密码默认为"123456"，学生首次登录后应修改密码

#### 6.1.2 分配导师

- **操作步骤**：
  1. 进入学生管理页面
  2. 点击学生列表中的"分配导师"按钮
  3. 在弹出的导师选择器中选择导师
  4. 点击"确认"按钮

- **注意事项**：
  - 一个学生只能有一个指导导师
  - 一个导师可以指导多名学生

### 6.2 课题进度

#### 6.2.1 提交进度

- **操作步骤**：
  1. 进入课题进度页面
  2. 点击"提交进度"按钮
  3. 填写进度信息：标题、内容、完成度、遇到的问题、下一步计划
  4. 点击"提交"按钮

- **注意事项**：
  - 进度提交后状态为"待审阅"
  - 学生可以在导师反馈前编辑已提交的进度

#### 6.2.2 导师反馈

- **操作步骤**：
  1. 进入待审进度页面
  2. 点击进度列表中的进度
  3. 查看进度内容
  4. 填写反馈意见，选择评分和是否通过
  5. 点击"提交反馈"按钮

- **注意事项**：
  - 反馈提交后，学生会收到通知
  - 导师可以多次编辑反馈

### 6.3 内容管理

#### 6.3.1 新闻管理

- **操作步骤**：
  1. 进入内容管理页面
  2. 点击"新闻管理"
  3. 点击"添加新闻"按钮
  4. 填写新闻信息：标题、内容、分类、封面图
  5. 点击"发布"按钮

- **注意事项**：
  - 新闻发布后会在首页显示
  - 可以编辑和删除已发布的新闻

#### 6.3.2 成果管理

- **操作步骤**：
  1. 进入内容管理页面
  2. 点击"成果管理"
  3. 点击"添加成果"按钮
  4. 填写成果信息：标题、描述、类型、作者、年份、链接、封面图
  5. 点击"提交"按钮

- **注意事项**：
  - 成果类型包括：论文、项目、获奖、专利
  - 可以编辑和删除已添加的成果

## 7. 常见问题

### 7.1 登录问题

**问题**：无法登录系统
**解决方案**：
- 检查用户名和密码是否正确
- 检查网络连接是否正常
- 联系管理员重置密码

### 7.2 权限问题

**问题**：无法访问某些页面
**解决方案**：
- 检查用户角色是否有相应的权限
- 联系管理员获取权限

### 7.3 提交进度问题

**问题**：无法提交进度
**解决方案**：
- 检查网络连接是否正常
- 检查填写的信息是否完整
- 联系管理员解决

### 7.4 反馈问题

**问题**：导师无法提交反馈
**解决方案**：
- 检查网络连接是否正常
- 检查填写的反馈信息是否完整
- 联系管理员解决
