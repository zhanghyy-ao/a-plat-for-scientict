## 一、系统架构概览

### 技术栈
- **前端**: React + TypeScript + Vite + Tailwind CSS + Framer Motion
- **后端**: Flask + Flask-RESTX + SQLAlchemy + JWT
- **数据库**: SQLite (开发) / MySQL (生产)

### 用户角色
1. **管理员 (admin)**: 系统管理、用户管理、内容管理
2. **导师 (mentor)**: 学生指导、进度审核、消息沟通
3. **学生 (student)**: 进度提交、资源学习、笔记管理
