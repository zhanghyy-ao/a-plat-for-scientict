# 登录说明

## 后端启动

1. 打开命令行，进入 `backend` 目录
2. 运行以下命令启动后端：
   ```
   python app.py
   ```

## 默认登录账号

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 管理员 | admin | admin123 |
| 导师 | mentor1 | mentor123 |
| 学生 | student1 | student123 |

## 常见问题

### 导师账号无法登录
如果导师账号无法登录，请尝试：

1. 确认后端服务已正常启动（运行在 http://localhost:5000）
2. 检查数据库文件是否存在：`backend/instance/lab_management.db`
3. 如果数据库有问题，可以删除数据库文件并重启后端，系统会自动重新创建
4. 查看后端控制台输出的调试信息

### 数据库重置
如需重置数据库：
- 删除 `backend/instance/lab_management.db` 文件
- 重启后端服务

## 功能说明

- **管理员**：可以管理学生账号、导师账号、内容管理
- **导师**：可以查看和管理自己的学生，审核进度报告
- **学生**：可以查看个人信息，提交进度报告，管理待办事项等
