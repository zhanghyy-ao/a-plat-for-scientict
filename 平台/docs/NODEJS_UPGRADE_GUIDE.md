# Node.js 升级指南

## 当前状态
- 当前Node.js版本: v18.20.8
- 所需版本: 20.19+ 或 22.12+

## 升级步骤

### 方法一：从官网下载安装包
1. 访问 Node.js 官网：https://nodejs.org/zh-cn/
2. 点击"下载"按钮，选择最新的 LTS 版本（长期支持版本）
3. 运行下载的安装程序
4. 按照安装向导的提示完成安装
5. 安装完成后，重启计算机

### 方法二：使用 nvm-windows（推荐）
1. 下载 nvm-windows 安装包：https://github.com/coreybutler/nvm-windows/releases
2. 运行安装程序，按照提示完成安装
3. 打开新的命令提示符窗口，运行以下命令：
   ```
   nvm install 20.19.0
   nvm use 20.19.0
   ```
4. 验证安装：
   ```
   node --version
   ```

## 验证升级
升级完成后，打开命令提示符窗口，运行以下命令验证版本：
```
node --version
```

如果显示的版本是 20.19+ 或 22.12+，则升级成功。

## 启动前端服务
升级完成后，可以运行以下命令启动前端服务：
```
npm run dev
```

前端服务将运行在 http://localhost:5173

## 注意事项
- 升级前请确保保存所有正在进行的工作
- 升级过程中可能需要重启计算机
- 如果遇到权限问题，请以管理员身份运行命令提示符
